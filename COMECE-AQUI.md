## 🆘 Precisa de Ajuda?

### Problemas Comuns

**❌ Erro ao instalar**
```bash
rm -rf node_modules package-lock.json
npm install
```

**❌ Porta em uso**
```bash
npm run dev -- --port 3000
```

**❌ Página em branco**
- Verifique o console (F12)
- Recarregue a página (Ctrl+F5)
